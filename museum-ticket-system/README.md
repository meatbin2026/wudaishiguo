# 博物馆特展门票购买系统

一个纯前端的简易购票系统，打开页面即可使用。

## 功能

- 特展选择、日期选择、场次选择
- 成人/学生/儿童票数量选择与实时金额计算
- 实时余票展示（按特展 + 日期 + 场次）
- 表单校验（姓名、手机号、邮箱格式）
- 下单后生成订单号并保存本地记录
- 订单列表展示、退票回补库存
- 一键清空订单与库存记录

## 使用

1. 直接在浏览器打开 `index.html`。
2. 也可以在该目录启动静态服务器后访问。

## 存储说明

系统使用浏览器 `localStorage` 保存数据：

- `museum_ticket_orders`：订单列表
- `museum_ticket_sold`：库存占用（已售票）

清除浏览器站点数据后，订单与库存会恢复初始状态。
